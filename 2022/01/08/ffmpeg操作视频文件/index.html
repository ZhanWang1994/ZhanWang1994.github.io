ffmpeg操作视频文件

Posted on 2022-01-08 Edited on 2022-02-26

JAVA通过命令行调用ffmpeg处理视频文件，需要先自行安装ffmpeg
代码示例：

代码等同于 ffmpeg -i input.mp4 output.flv
1.视频码率调整
ffmpeg -i input.avi -b:v 1500k output.avi
2.视频格式调整
ffmpeg -i input.avi output.mp4
3.视频尺寸调整
ffmpeg -i input.mpg -s 320x240 output.mp4
ffmpeg -i input.mpg -vf scale=320:240 output.mp4
1）等比例缩放
ffmpeg -i input.mpg -vf scale=iw05:ih0.5 output.mp4(等比例缩小到50%)
2）加黑边（宽高比不符合）
ffmpeg -i src.avi -vf pad=640:480:0:40:black -y test_pad.avi
若源视屏尺寸为1280800 目标尺寸为640480
那么先等比例缩放640400 黑边宽度为（480-400）/2=40
black标识黑边 可以不写，默认为黑色
4.视频部分替换成图片
1)把图片转成视频
ffmpeg -r 25 -loop 1 -i G://ffmpeg-demo/test.jpg -pix_fmt yuv420p -vcodec libx264 -b:v 600k -r:v 25 -preset medium -crf 30 -s 1280720 -vframes 250 -r 25 -t 3 G://ffmpeg-demo/test.mp4

ffmpeg -r 25 -loop 1 -i G://ffmpeg-demo/test.jpg -pix_fmt yuv420p -vcodec libx264 -b:v 600k -r:v 25 -preset medium -crf 30 -s 1280*720 -vframes 250 -r 25 -t 3 G://ffmpeg-demo/图片转视频.mp4
2)转化图片视频以及截取源视频，准备合并
ffmpeg -i test.mp4 -c copy -bsf:v h264_mp4toannexb -f mpegts 1.ts
ffmpeg -ss 00:00:00 -t 12 -i 源视频.mp4 -c copy -bsf:v h264_mp4toannexb -f mpegts 2.ts
3）合并视频
ffmpeg -i “concat:2.ts|1.ts” -c copy -bsf:a aac_adtstoasc -movflags +faststart 合并.mp4

ffmpeg –i G://ffmpeg-demo/源视频.mp4 -acodec copy-vcodec copy -vf “movie=G://ffmpeg-demo/底部.jpg[watermark];[in][watermark]overlay=10:10:1[out]” G://ffmpeg-demo/加完水印.mp4

ffmpeg -y -i 源视频.mp4 -vf “movie=左上.jpg []; [in][logo] overlay=1:1 [out]” output.mp4

ffmpeg -y -i 源视频.mp4 -i 左上.jpg -i 底部.jpg -filter_complex “[1:v]scale=100:100[img1];[2:v]scale=720:100[img2];[0:v][img1]overlay=0:0[bkg];[bkg][img2]overlay=0:400” output2.mp4

ffmpeg -i 源视频.mp4 -i 左上.jpg -i 底部.jpg -loop 1 -t 4.5 -i 片尾.jpg -filter_complex “ [1:v]scale=100:100[img1];[2:v]scale=720:100[img2];[0:v][img1]overlay=0:0[bkg];[bkg][img2]overlay=0:400[bkg];[bkg][3:v]scale=w=480:h=360[out_5];[out_4][out_5]concat “ -acodec copy -vcodec libx264 -b:v 500k -threads 4 -thread_type frame -preset faster -tune zerolatency -y test.mp4

5.令生成的MP4视频允许边下边放
教程：
https://blog.csdn.net/zhoubotong2012/article/details/79346363

需要使用工具将meta信息移动到视频头部
